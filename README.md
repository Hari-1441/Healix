# HEALIX  
### Intelligent Healthcare Management System

HEALIX is a cloud-integrated healthcare platform designed to improve medication adherence and enable efficient patient data management through real-time tracking and user-centric features.

---

## 🚀 Features
- Real-time medication reminders and tracking  
- Voice-based interaction for accessibility  
- Health notes and patient data logging  
- AI-powered chat support  
- Interactive monitoring dashboard  

---

## 🧠 System Overview
HEALIX is built as a real-time healthcare system integrating a lightweight frontend with a cloud-based backend to ensure seamless data handling and user interaction.

- Frontend: Streamlit  
- Backend: Firebase (real-time database)  
- AI Integration: GROQ-based assistant  
- Architecture: Cloud-connected, real-time system
